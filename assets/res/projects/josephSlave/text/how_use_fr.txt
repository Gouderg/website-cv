Pour pouvoir utiliser le bot, il faut installer les différents paquets: npm, curl, nodejs, ffmpeg
Avec npm, il faut installer les paquets suivants: discord.js, node-opus, youtube-audio-stream, xmlhttprequest

Ensuite, il faut créer un fichier config.json avec la structure suivante:

{
"token" : "insert_token",
	"cleNasa" : "insert_token",
	"cleOpenWeather" : "insert_token",
	"cleJoke" : "insert_token"
}

Enfin, il faut lancer le bot avec la commande suivante:
<span class="code">node index.js</span>

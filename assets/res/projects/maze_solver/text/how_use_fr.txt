Pour utiliser ce programme, il faut SDL1.2, SDL1.2-mixer, SDL1.2-image et C.
Pour ubuntu ce sont ces paquets: libsdl1.2debian libsdl1.2-dev, libsdl-mixer1.2, libsdl-mixer1.2-dev, libsdl-image1.2, libsdl-image1.2-dev

Et ensuite il faut compiler le code avec la commande: <span class="code">make</span>
Enfin, on execute le code avec la commande: <span class="code">./Robot.exe</span>
